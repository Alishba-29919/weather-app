# Weather App

A clean and functional weather application that provides real-time weather updates using the OpenWeatherMap API.

## 🚀 Key Features
- **Real-time Data:** Accurate temperature and weather conditions fetching.
- **Dynamic Backgrounds:** UI updates based on the current weather (Warm/Cold).
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop.
- **Search Functionality:** Get weather information for any city worldwide.

## 🛠️ Built With
- HTML5 & CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
